1주차 (Day 01-07) | 현재형 기초
  핵심 조사: に / を / で / が
  핵심 동사: いきます / かえります / します / たべます / のみます / かいます / います / あります
  핵심 의문사: いつ / どこ / だれ / なに / なんじ

2주차 (Day 08-14) | 과거형 + 권유 + 희망
  과거형: ます → ました / い형용사 → かった
  권유형: ます → ましょう
  희망형: 동사 어간 → たいです

3주차 (Day 15-21) | テ형 활용
  요청:  テ形 + ください
  연결:  テ形 + 동사
  허가:  テ形 + もいいですか
  금지:  テ形 + はいけません

4주차 (Day 22-30) | 기본형 + 보통체
  능력:  基本形 + ことができます
  전후:  基本形 + まえに / 過去形 + あとで
  추측:  基本形 + とおもいます
  반말:  정중체 → 보통체 전환
  변화:  〜ようになりました
